# Sai Gowtham - Personal Portfolio

A professional, high-performance portfolio website built with:
- **Frontend**: React, Vite, Tailwind CSS v4, Framer Motion
- **Backend**: Node.js (Express) - optimized for Vercel Serverless
- **Design**: "Future of AI" aesthetic (Neon Cyan/Purple, Dark Mode)

## Project Structure
```
/
├── api/                 # Express backend (Serverless entry point)
├── src/
│   ├── components/      # UI Components (Hero, About, Skills, etc.)
│   ├── assets/          # Static assets
│   ├── App.jsx          # Main application layout
│   └── index.css        # Tailwind V4 & Global styles
├── vercel.json          # Vercel deployment config
└── vite.config.js       # Vite config (incl. API proxy)
```

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Locally (Dev Mode)**
   This starts the Vite frontend server.
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```
